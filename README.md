# Projeto - Tela de login

#### Descrição

Este projeto foi desenvolvido como parte prática da disciplina de **Programação Front-End**, no campus da **UniCesumar Londrina**. Sob a orientação do professor [Leonardo Rocha](https://github.com/leonardossrocha), exploramos os conceitos fundamentais de **Frameworks**, utilizando o **Bootstrap** e seus componentes. Além disso, realizamos toda a configuração do ambiente de desenvolvimento, integrando ferramentas essenciais para o fluxo de trabalho de um desenvolvedor.

### Ferramentas Utilizadas 

- HTML5
- CSS3
- Bootstrap
- Git

### Tecnologias e Fluxo de Trabalho

* **Live Server:** Extensão do VS Code que atualiza o navegador automaticamente a cada alteração salva.
* **Bootstrap (via CDN):** Utilizamos a CDN (Content Delivery Network) para integrar o framework de forma rápida, carregando arquivos CSS e JavaScript diretamente de servidores externos otimizados, sem a necessidade de baixar os arquivos localmente.
* **Git & GitHub:** Essenciais para o versionamento do código e documentação profissional do projeto.

#### FAQ e Navegação
Desenvolvemos uma seção de **FAQ (Perguntas Frequentes)** funcional. O objetivo foi praticar a lógica de navegação entre múltiplos arquivos HTML, onde o usuário é redirecionado para outra página ao interagir com as perguntas.

#### Comandos Essenciais de Git
Durante as atividades, utilizamos os comandos fundamentais para manter o projeto organizado:
* `git status`: Para monitorar o estado atual dos arquivos.
* `git add`: Para preparar as alterações para o próximo registro.
* `git commit`: Para salvar permanentemente as alterações no histórico com uma mensagem descritiva.
* `git push`: Para subir o projeto local para o repositório remoto no GitHub.
